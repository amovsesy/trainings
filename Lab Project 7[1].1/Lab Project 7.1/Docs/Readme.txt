1. Controls
-----------

Key Controls :

    Escape         - Exit

    Cursor Up      - Move Camera Forward
    Cursor Down    - Move Camera Backward
    Cursor Left    - Strafe Camera Left
    Cursor Right   - Strafe Camera Right
    Page Up        - Move Camera Up
    Page Down      - Move Camera Down

    Shift + Return - Toggle between windowed / fullscreen modes

Mouse Controls :
    
    Left Button    - Use Mouse Look
    Left + Right   - Roll camera (or, Lean in FPS mode)
