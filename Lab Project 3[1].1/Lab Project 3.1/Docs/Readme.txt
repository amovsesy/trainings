1. Introduction
---------------

In this demonstration we start to introduce you to the concept of vertex
buffers and also how to create, fill and render the simple non-indexed variety.

Although once again this demonstration makes use of extremely simple geometry
(our baseline two spinning cubes *yawn* ...ed) once we have learned to take
advantage of these types of objects, and how we can make use of the hardware
transformation and lighting features of the graphics hardware, we can start to
create some pretty interesting, and not to mention FAST, rendered scenes.

2. General Usage
----------------

The application is a low interactivity demo which simply demonstrates both the
enumeration for supported hardware features, and the creation, filling, and
rendering of a simple vertex buffer. 

3. Controls
-----------

Escape         - Exit
Cursor Left    - Strafe Camera Left
Cursor Right   - Strafe Camera Right
Shift + Return - Toggle between windowed / fullscreen modes
